# Công cụ quản lý
Microsoft có rất nhiều công cụ nên họ chia ra các trang admin để quản lý cho từng công cụ.
Microsoft admin center https://admin.microsoft.com/ là trang tổng hợp các trang admin giúp quản lý toàn bộ các ứng dụng của microsoft.
Microsoft admin center cung 1 unified management interface and console cho admin để quản lý cả Azure AD và M365 (đương nhiên user chỉ access được vào M365 sau khi mua subscription M365 rùi), và user chỉ vào được trang nếu là được cấp quyền bởi admin.
Vd team admin center là 1 phần của admin center nhưng chỉ quản lý chuyên cho Microsoft team.

Thông thường họ k để 1 admin quản lý tất cả mà tạo ra nhiều tk admin ở các tenant con khác nhau, gọi là các dedicated admin. Nhưng ở đây vc chia tenant ra k có ý nghĩa nh về logic mà chỉ là tách các phần độc lập của organization ra mà thôi.

Azure Portal có thể coi là trang admin center của azure. Nhưng Azure rất lớn và liên quan đến nhiều thứ nên đôi khi ta thấy có cái bị overlap dịch vụ nhau và ta có thể thích quản lý ở đâu cũng được, VD Azure Portal và Microsoft Admin Center hay Microsoft Entra Id. 
=> Đó là lý do ta thấy danh sách user ở admin center chung, ở microsof entra id, ở azure portal tab user, thật ra là bị overlap dịch vụ thoi



# API
Với số lượng ứng dụng lớn như v, API của microsoft cũng k central. Có nhiều loại API:
- Microsoft Graph API: A single REST API that provides access to resources across multiple Microsoft services, including Office 365, OneDrive, SharePoint, and more, nó chứa các thông tin chung
- Azure Management API: A REST API that provides programmatic access to Azure resources, such as virtual machines, storage accounts, and networking resources. Docs thằng này có Sku.isTrial mà éo biết lấy ra sao
- Azure Active Directory Graph API: A REST API that provides access to user and group data in Azure Active Directory, including user profiles, group membership, and more.
- Office 365 Management Activity API: A REST API that provides access to the Office 365 audit log, which records user and administrator activity across various Office 365 services.
- Office 365 Service Communications API: A REST API that provides information about the status of Office 365 services, including licensing information.
- Microsoft Power Platform APIs: A set of REST APIs that provide access to Power Apps, Power Automate, and Power BI resources, including data connectors, flows, and visualizations.
- Microsoft Dynamics 365 APIs: A set of REST APIs that provide access to data and resources in Microsoft Dynamics 365, including customer and sales data, customer service data, and more.
- Microsoft Translator API: A REST API that provides programmatic access to the Microsoft Translator service, which can be used to translate text and speech between different languages.
- Bing Search API: A REST API that provides access to the Bing search engine, which can be used to search for web pages, images, videos, and news articles.
- Microsoft Cognitive Services APIs: A set of REST APIs that provide access to various cognitive services, such as speech recognition, image recognition, and natural language processing.
